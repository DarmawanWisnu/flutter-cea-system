// turbo-all

Semua perintah dijalankan dari folder `apps/mobile/`

1. Jalankan unit tests di folder test/
```bash
cd apps/mobile && flutter test test/
```

2. Jalankan integration tests di folder integration_test/ (butuh emulator/device)
```bash
cd apps/mobile && flutter test integration_test/
```
